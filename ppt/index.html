System Architecture · 系统架构

MULTI-AGENT · THREE LAYERS

三层架构,协同调度

01 · ORCHESTRATOR

编排调度层

统一调度三个专职 Agent,合并去重审计结果,输出结构化报告.支持串行流水线执行.

02 · AGENTS

专职 Agent 层

Security Agent / Refactor Agent / Similarity Agent,各司其职,独立运行.

03 · TOOLS

工具与模型层

bandit 静态扫描 / MiMo LLM 推理 / TF-IDF 相似度检索 / Python AST 解析.

CORE AGENTS · 核心组件

四大核心模块

— 01 / ORCHESTRATOR

编排调度器

统一调度三大 Agent,合并去重结果,输出结构化审计报告.支持流水线模式.

— 02 / SECURITY

安全审计 Agent

基于 bandit 静态分析引擎,检测 SQL 注入、XSS、硬编码密钥等 OWASP Top 10 漏洞.

— 03 / REFACTOR

重构分析 Agent

LLM 智能推理 + 规则引擎双模式:MiMo 大模型深度分析代码质量,AST 解析提供结构化建议.

— 04 / SIMILARITY

相似度检索 Agent

TF-IDF 向量化 + 余弦相似度,从历史修复案例库中检索最相似的解决方案.
